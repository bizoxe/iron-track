Custom JSON Response¶

class app.lib.json_response.MsgSpecJSONResponse(content, status_code=200, headers=None, media_type='application/json', background=None)[source]¶

Bases: Response

High-performance JSON response using msgspec for serialization.

__init__(content, status_code=200, headers=None, media_type='application/json', background=None)[source]¶

Initializes the custom response class.

Parameters:

  • content – The content to be serialized.

  • status_code (int) – HTTP status code.

  • headers (Mapping[str, str] | None) – HTTP headers.

  • media_type (str) – Media type (default: application/json).

  • background (BackgroundTask | None) – Background task to run after the response is sent.

render(content)[source]¶

Serialize the content using the msgspec encoder.

The content is first passed through fastapi.encoders.jsonable_encoder to handle standard FastAPI types (e.g., Pydantic models).

Parameters:

content – The data to serialize.

Returns:

Serialized JSON content as bytes.

Return type:

bytes
